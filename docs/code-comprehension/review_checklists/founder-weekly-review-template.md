# Founder Weekly Review Template
Use this to keep leadership visibility high without heavy process.

1. **System Snapshot (2 sentences)** – What shipped or changed this week?
2. **Critical Paths Status** – Bullet status for:
   - Financial data pipeline
   - Platform build & datasets
   - Supabase live data
   - Intake funnels
   - Monitoring/alerts
3. **Freshness & Data Truth** – When were `/data/financials.json`, `cranegenius_*`, and Supabase feeds last updated? Note any unknowns.
4. **Silent Failure Watchlist** – Any dashboards or feeds currently in fallback mode? Explain.
5. **Incidents / Near Misses** – Summarize outages, stale data, or security concerns.
6. **Next Week’s Top 3 Fixes** – Concrete deliverables tied to the roadmap.
7. **CTO Talking Points** – One paragraph founder-ready summary for investor/customer conversations.
