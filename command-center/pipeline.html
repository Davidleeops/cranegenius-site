Runbook & schedule

Run from repo root. Full runbook: docs/mvp_pipeline_runbook.md. Cron example: docs/cron_mvp_pipeline.example.

Run pipeline
python3 scripts/run_mvp_pipeline.py
Refresh this dashboard
python3 tools/encrypt_command_center_data.py
Schedule daily (cron)
0 6 * * * cd /path/to/cranegenius_repo && python3 scripts/run_mvp_pipeline.py >> runs/mvp_pipeline/cron.log 2>&1
Edit path, then crontab -e and paste the line.

python3 tools/encrypt_command_center_data.py
