PIPELINE , Data Processing Layer
8-stage Python pipeline. Runs locally + via GitHub Actions.
LIVE
Stage 1 , Ingest
scrape_permits_public.py pulls all source APIs in parallel with timeout controls.
src/ingest.py · Python 3.11
Stage 2 , Normalize
All rows mapped to canonical signal schema. City, state, family, confidence score.
src/parse_normalize.py
Stage 3 , Score
Crane relevance filter + value threshold. Residential noise removed.
scripts/signal_scoring.py
Stage 4 , Domain Enrich
Company name → domain guess → contact page scrape.
src/domain_enricher_claude.py · Claude API
Stage 5 , Contact Mine
Extract emails from contact pages. Generate firstname@, info@, estimating@ patterns.
src/contact_page_finder.py
Stage 6 , Verify
MillionVerifier API checks every email. Keep valid + catchall only.
MillionVerifier API · $0.001/email
Stage 7 , Export
CSV output: sender_ready_hot.csv, sender_ready_warm.csv, catchall_review.csv
src/exporter.py · Google Sheets
Stage 8 , Intelligence
Project scoring, delay pressure, constraint scores. Feeds Catch-Up Radar.
scripts/build_catch_up_rank.py
